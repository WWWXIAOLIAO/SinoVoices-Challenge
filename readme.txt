Example2
Background and Motivation
随着大语言模型技术的快速迭代与普及，第一视角可穿戴设备迎来爆发式发展，其中AI眼镜凭借其便携性、沉浸式交互优势，已成为人机交互领域的核心载体之一。作为典型的第一视角可穿戴设备，AI眼镜具备天然的多模态交互能力，可实时采集四通道音频数据与视频数据，打破了传统单一模态交互的局限，推动人机交互从“被动响应”向“主动协作”升级，在日常办公、社交沟通、户外出行等场景中得到广泛应用。
然而，在真实场景的工程化部署中，智能眼镜的音视双模态交互仍面临三大核心瓶颈，严重制约其实际应用价值与用户体验，这也成为本次竞赛发起的核心动机：
1. 噪声环境的鲁棒性瓶颈
AI眼镜作为随身穿戴设备，常处于高动态复杂物理环境中，其采集的音频信号容易受到非稳态噪声（如环境音乐、交通轰鸣、强风噪）及佩戴者自身肢体运动产生的摩擦干扰。此类噪声具有高分贝、非人声、频谱覆盖范围广的显著特征，对传统单通道降噪算法的降噪效果与实时性提出极高挑战，极易导致语音识别与理解精度大幅下降。
2. 社交环境的拾音精准度瓶颈
在餐厅、聚会、公共场所等真实社交场景中，周围人群的语音会形成复杂的社交噪声，即典型的“鸡尾酒会效应”。此时，系统需充分利用麦克风阵列的物理空间几何线索，实现目标说话人的精准锁定与语音提取，这也是现有单通道或低通道可穿戴设备难以高效解决的关键问题。
3. 复杂交互的语义理解瓶颈
真实场景下的口语交互指令具有高度非结构化特征，包含大量文本指代、多轮嵌套及潜台词（如暗示、双重否定、反讽）。此外，交互过程中常出现基于实时环境变量（如天气变化、场景变化）导致的决策修正，这就要求大模型具备长上下文记忆回溯与动态逻辑推理能力，而现有模型在该方面的能力仍有明显不足。

值得注意的是，现有公开数据集（如WearVox）在数据多样性、噪声真实性及多模数据完整性上存在局限，难以充分展现智能眼镜场景下的模型性能局限。本次竞赛依托商业AI眼镜设备，在多种真实场景下录制四通道音视频数据，通过与WearVox数据集的对比，更能凸显参赛算法的实际优势与工程价值。

Core Objectives/Challenge Overview
本次竞赛提出EgoAIG (Egocentric Interaction on AI Glasses Challenge)，是一个基于真实对话录制的基准竞赛，覆盖人机交互、双人对话、多人会议真实场景。所有录音均通过AI眼镜的麦克风在真实生活场景中采集，数据集为每位说话人提供多段注册语音，可实现稳健的注册感知条件适配。
参赛队伍需构建一套多模态交互系统，该系统需接收：• 真实场景下的音视混合信号，• 目标说话人的注册信息，输出提取后的目标说话人信号及对应的语义理解结果。
本次竞赛设置三个赛道，分别对应不同的真实交互场景，核心目标是评估参赛系统在真实复杂环境下的语音识别及语义理解能力，具体分为：
1. 人机对话赛道：评估系统在音视双模态人机交互场景下的声学鲁棒性与语义理解能力；
2. 双人对话赛道：评估系统在双人交流场景下的多说话人分离、识别与语义理解能力；
3. 多人会议赛道：评估系统在多说话人、高干扰会议场景下的声学处理与语义理解能力。
这些赛道共同构成了真实场景下智能眼镜音视双模态交互的实用基准，既评估可部署的实时交互系统，也检验高性能的离线处理方法，推动相关技术的工程化落地。

Task Setup and Evaluation Methodology
本次竞赛基于商业AI眼镜设备开展，录制200小时真实数据，并进一步划分为训练集与测试集，确保评估的公平性、真实性与针对性。竞赛根据交互场景复杂度分为3个Track，每个Track均包含识别与理解两个核心任务，具体设置如下：
Track 1: 人机对话场景
核心场景：模拟AI眼镜用户与设备的日常人机交互，涵盖视觉问答与音频问答两类内容，包含单轮对话与多轮对话两种交互形式，重点考察模型在复杂场景下的声学适配性与语义理解能力。
1. 识别任务
评估目标：考察模型在复杂声学场景下的语音采集与识别稳健性，重点覆盖场景多样性、说话人年龄性别多样性及说话音量梯度差异三大维度。
评估指标：语音识别性能采用词错误率（WER）进行量化评估，WER越低，表明声学适配性与识别精度越高。
2. 理解任务
评估目标：考察模型对人机对话中语义信息的理解能力，包括对视觉问答、音频问答的语义解析与答案匹配能力。
评估方式：采用选择题打分形式，根据模型对题目答案的匹配准确率进行评分，准确率越高，表明语义理解能力越强。
Track 2: 双人对话场景
核心场景：模拟两人面对面交流场景，仅涉及音频信息，重点考察模型对多说话人语音的分离、识别与语义理解能力。
1. 识别任务
评估目标：考察模型对双人对话中目标说话人的识别、语音转录能力，克服两人语音重叠、背景干扰带来的挑战。
评估指标：采用转录条件词错误率（tcpWER）作为核心指标，同时结合说话人识别准确率、语音转录完整性进行综合评判，tcpWER越低、识别与转录准确率越高，声学性能越优。
2. 理解任务
评估目标：考察模型对双人对话核心语义的理解能力，准确捕捉对话中的关键信息与逻辑关系。
评估方式：采用选择题打分形式，根据模型对对话语义相关题目的答案匹配准确率进行评分，量化模型的语义理解精度。
Track 3: 多人会议场景
核心场景：模拟不同规模、不同场景的多人会议场景，仅涉及音频信息，重点考察模型在多说话人、高干扰环境下的声学处理与语义理解能力。
1. 识别任务
评估目标：考察模型对多人会议中目标说话人的识别、语音转录能力，克服语音重叠、背景干扰带来的挑战。
评估指标：采用转录条件词错误率（tcpWER）作为核心指标，同时结合说话人识别准确率、语音转录完整性进行综合评判，tcpWER越低、识别与转录准确率越高，声学性能越优。
2. 理解任务
评估目标：考察模型对多人会议内容的语义提炼、关键信息提取能力，检验模型在长上下文、多角色交互场景下的逻辑推理与信息匹配能力。
评估方式：采用选择题问答形式，基于会议音频内容设计针对性问题，根据模型对问题答案的匹配准确率进行量化评分，以此评估模型对会议核心语义的理解与信息提取精度。

Background and Motivation

随着大语言模型技术的快速迭代与普及，第一视角可穿戴设备迎来爆发式发展，其中AI眼镜凭借其便携性、沉浸式交互优势，已成为人机交互领域的核心载体之一。作为典型的第一视角可穿戴设备，AI眼镜具备天然的多模态交互能力，可实时采集四通道音频数据与视频数据，打破了传统单一模态交互的局限，推动人机交互从“被动响应”向“主动协作”升级，在日常办公、社交沟通、户外出行等场景中得到广泛应用。然而，在真实场景的工程化部署中，智能眼镜的音视双模态交互仍面临三大核心瓶颈，严重制约其实际应用价值与用户体验，这也成为本次竞赛发起的核心动机：首先，在高动态物理环境中，非稳态噪声（如交通、风噪及设备摩擦）会导致系统声学鲁棒性骤降；其次，在“鸡尾酒会”等多说话人社交场景下，受限于眼镜端低功耗微型麦克风阵列，系统对目标拾音与空间分离的精度大打折扣；最后，面对真实世界中高度非结构化、充满潜台词与动态环境变量的开放口语指令时，现有模型在复杂语义理解与长上下文逻辑推理上存在明显断层。与此同时，现有公开数据集（如WearVox、MMCSG）在声学场景多样性、噪声干扰真实度以及多模态数据完整性上存在明显局限，难以真实反映和全面评估智能眼镜在极限场景下的模型性能边界。因此，我们发起XXX竞赛，旨在填补评测空白，构建极具工程应用价值的评估基准，激发社区在端侧抗噪、空间拾音及多模态语用推理等方向的底层创新。
Challenge Overview
本次挑战赛将发布首个基于商业级AI智能眼镜采集的大规模真实交互数据集。该数据集包含200小时的四通道（4-channel）音视频序列，完全在真实生活场景中录制采集。为确保评估的公平性与全面性，数据覆盖了多梯度环境噪声、佩戴者自噪声，以及丰富的说话人画像与多人重叠社交场景。数据集将被严谨划分为训练集与测试集，并为每位说话人提供多段先验注册信息。
参赛队伍需构建一套端到端的多模态交互系统。该系统需同时接收“真实场景下的音视混合强干扰信号”与“目标说话人的注册信息”，最终输出提取并转录后的目标说话人纯净语音，以及深层语义理解结果。
依据交互场景的复杂度,本次竞赛设置三个赛道，分别对应不同的真实交互场景，核心目标是评估参赛系统在真实复杂环境下的语音识别及语义理解能力，具体分为：
1. 人机对话赛道：评估系统在音视双模态人机交互场景下的声学鲁棒性与语义理解能力；
2. 双人对话赛道：评估系统在双人交流场景下的多说话人分离、识别与语义理解能力；
3. 多人会议赛道：评估系统在多说话人、高干扰会议场景下的声学处理与语义理解能力。
这些赛道共同构成了真实场景下智能眼镜音视双模态交互的实用基准，既评估可部署的实时交互系统，也检验高性能的离线处理方法，推动相关技术的工程化落地。
Evaluation Methodology
为全面衡量参赛系统在第一视角复杂环境下的端到端交互能力，所有赛道均统一设置“识别”与“理解”两项联合评估任务。识别任务要求系统利用注册信息，在非稳态噪声与多说话人重叠干扰下精准提取并转录目标语音；其中 Track 1 采用词错误率（WER）进行量化，而包含密集语音交锋的 Track 2 与 Track 3 则采用转录条件词错误率（tcpWER）以综合评判说话人分离与转录精度。理解任务则超越底层转录，聚焦于多模态对齐（如音视问答）与长上下文核心信息提炼，统一采用基于真实语境构建的多项选择问答准确率（Multiple-Choice QA Accuracy）进行客观评分。
